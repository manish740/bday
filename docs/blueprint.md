# **App Name**: Natal Reminders

## Core Features:

- User Authentication: Secure user signup and login to manage personal birthday contacts.
- Birthday Contact Management: Add, edit, and delete birthday entries including names and dates for your friends and family.
- Upcoming Birthdays Dashboard: A clear, intuitive dashboard displaying upcoming birthdays sorted by date.
- AI Personalized Wish Tool: Generate unique birthday messages tailored to the relationship type and desired tone (e.g., formal, casual, humorous).
- Wish Preview & Copy: Allow users to preview AI-generated wishes and easily copy them for use in other platforms.

## Style Guidelines:

- Primary color: A sophisticated and warm dusty rose (#D18CA9), chosen to evoke feelings of celebration and personal connection. This shade is elegant yet friendly.
- Background color: A very light, desaturated variation of the primary color (#F9EDF1) to maintain harmony and provide a serene canvas for content, fitting for a light color scheme.
- Accent color: A deep, rich plum (#6D2680), providing a strong and contrasting element that complements the primary rose, used for key actions and highlights.
- Headline font: 'Belleza' (humanist sans-serif) for an elegant, artistic, and modern aesthetic. Body text font: 'Alegreya' (humanist serif) to provide a comfortable, refined reading experience for generated messages and descriptions.
- Minimalist line-art icons related to celebrations, gifts, and calendar events, designed to be subtle and enhance the overall elegant feel.
- A clean, card-based layout for birthday entries, ensuring high readability and easy navigation. Focus on responsive design for both desktop and mobile viewing.
- Subtle fade-in and slide-in animations for new content loading or dashboard updates, enhancing user interaction without distracting from the core purpose.
